# Product

## Vision

DisasterPH is a Philippines-focused disaster awareness app built around a live national map. The product should feel like an operations screen for monitoring active hazards, not a report viewer or generic admin console.

## Target Users

- Local government disaster risk reduction offices
- Emergency operations and response teams
- NGOs and humanitarian coordinators
- Journalists and civic monitoring groups
- Preparedness-focused public users

## MVP Scope

- Full-screen Philippines map as the primary interface
- Pulsing incident markers
- Desktop right-side panel for feed, details, stats, and source health
- Mobile bottom sheet for the same supporting information
- Filters by disaster type and region
- Selected incident details
- Source health and last-updated visibility
- Normalized multi-source incident model
- Initial alert engine behavior

## Out Of Scope

- Deep historical analytics
- User account management
- Full dispatch workflows
- Offline-first mode
- Advanced geofencing
- Multi-language support

## Product Principles

- Map first, text second
- Mobile first, not desktop squeezed smaller
- Compact, scannable, operational surfaces
- Data should look unified even when sources differ
- Source credibility and freshness should always be inspectable
- Build for future analytics without distorting the MVP
