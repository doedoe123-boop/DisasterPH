# MVP Scope

## In Scope

1. Full-screen Philippines-first monitoring UI
2. Glow pulse markers for active incidents
3. Active alert feed with compact cards
4. Selected incident detail panel
5. Filter chips by disaster type
6. Quick operational stats
7. Source health and last fetch timestamps
8. Mock normalized incident API response
9. Mobile bottom sheet behavior
10. Loading and empty states

## Out of Scope

- User accounts and RBAC
- Push notifications
- Historical analytics
- Multi-language support
- Geofenced subscriptions
- Full GIS layer management
- Dispatch tooling
- Offline mode

## MVP Success Criteria

- A user can identify active incidents and their locations quickly
- A user can distinguish official vs reference sources
- A user can filter the board without losing context
- Mobile and desktop flows both feel intentional
- The codebase is ready for real ingestion and persistence
