# Product Overview

## Vision

DisasterPH is a live disaster awareness dashboard designed for the Philippines. It should feel like an operations screen used to scan the national picture quickly, not a report viewer or admin table.

## Primary Users

- Local government operations centers
- Emergency response coordinators
- NGO monitoring teams
- Journalists and civic technology teams
- Preparedness-focused citizens

## Core User Jobs

- Scan where active incidents are happening in the country
- Identify which incidents need attention first
- Review the latest official and reference-source updates
- Understand source freshness before acting on information
- Filter fast by disaster type without leaving the map context

## Experience Principles

- Map dominates the interface
- Status must be scannable in under 10 seconds
- Details appear without losing geographic context
- Source credibility is visible, not hidden
- Mobile should preserve the command-center feel, not collapse into a plain feed

## Product Pillars

- Live national map
- Compact operational summaries
- Normalized multi-source event data
- Source-health transparency
- Mobile-first incident triage
